# 🧓 Бабушкиберпанк — генератор

Статический генератор для настольной ролевой игры **«Бабушкиберпанк»**.

Проект позволяет быстро нароллить:
- 👵 персонажа (бабушку с прошлым, вещчью и коронным номером)
- 📖 приключение (зацепка + 3 сцены)
- 🎶 частушки (d6 на любой случай жизни)

👉 Открыть генератор:  
https://prisioner.github.io/babushkyberpunk-generator

---

## 📚 Важно про оригинальную игру

Этот генератор основан на ролл-таблицах из книги правил **«Бабушкиберпанк»**.

⚠️ Все права на игру, тексты и оригинальный контент принадлежат авторам игры.  
Данный проект является фанатским инструментом и не заменяет книгу правил.

👉 Поддержать авторов и приобрести книгу:  
https://rpgbook.ru/babushkyberpunk

---

## ⚙️ Технологии

Проект реализован как полностью статическое приложение:

- React
- Vite
- TypeScript
- CSS (без UI-фреймворков)

Без:
- бэкенда
- API
- базы данных

---

## 🤖 Использование ИИ

ИИ применялся для:

---

## 🧩 Особенности

- Полностью client-side генерация
- Data-driven подход (все таблицы в JSON)
- Поддержка reroll отдельных элементов
- Минималистичный интерфейс без перегрузки
