# 🧓 Бабушкиберпанк — генератор

Ой, девоньки… докатились.  
Киберпанк уже и до нас добрался.

Это фанатский генератор для НРИ **«Бабушкиберпанк»**, чтобы быстро наскрести по сусекам:

- 👵 бабку (с коронным номером, прошлым и вещчью)
- 📖 приключение (чтобы было, куда ввязаться)
- 🎶 частушку (на всякий случай жизни)

👉 Вот тут всё работает, жми да крути:  
https://prisioner.github.io/babushkiberpunk-generator

## ⚙️ Из чего это слеплено

Без этих ваших серверов, всё по-простому:

- React
- Vite
- TypeScript
- CSS (руками, без ваших антов и прочего)

Без:
- API
- бэкенда
- баз данных

Всё работает прямо в браузере, как у людей.

## 🧩 Чем хороша эта штука

- Всё генерится прямо у тебя в браузере
- Никаких загрузок — щёлк и готово
- Можно перекручивать (reroll), пока не понравится
- Таблицы лежат в JSON — всё прозрачно

## 🚀 Как запустить у себя

Если вдруг захотел поковыряться:

```bash
npm install
npm run dev
```

Собрать:

```bash
npm run build
```
