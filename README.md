# Babushkiberpunk Generator

Статический генератор для настольно-ролевой игры «Бабушкиберпанк».

Проект предназначен как удобный цифровой инструмент для:
- генерации персонажа (бабушки);
- генерации приключения (зацепка + сцены);
- быстрого доступа к ролл-таблицам из книги.

---

## ✨ Возможности (MVP)

### 👵 Генератор бабушки
- Имя и отчество (d6)
- Прошлое (d66 с диапазонами)
- Коронный номер (d66 с диапазонами)
- Вещчь (d66 с диапазонами)
- Недостаток (d6)
- Распределение характеристик вручную (по правилам системы)
- Перегенерация:
    - всего персонажа
    - отдельных элементов

---

### 🎲 Генератор приключения
- Зацепка (d66)
- 3 сцены
- Каждая сцена:
    - место действия (d66)
    - киберпанк / технология (d66)
    - враги народа:
        - внешний вид (d6)
        - тип (d6)
        - подтип (d6, зависит от типа)

- Перегенерация:
    - всего приключения
    - отдельной сцены
    - отдельных элементов сцены

---

### 🎵 Дополнительно
