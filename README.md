# Babushkiberpunk Generator

Статический генератор для системы «Бабушкиберпанк».

## MVP
- Генератор бабушки (листа персонажа)
- Генератор приключения (зацепка + 3 сцены)
- Перегенерация целиком и по отдельным элементам
- Отображение используемых ролл-таблиц

## Tech
- React
- Vite
- TypeScript
- GitHub Pages
